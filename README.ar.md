# استوديو المنبر (Minbar Studio)
**مجموعة أدوات البث الإسلامي الاحترافية لبرنامج OBS Studio**

![Minbar Studio Banner](https://via.placeholder.com/1920x1080/0A192F/D4AF37?text=Minbar+Studio+-+Islamic+Broadcast+Toolkit)

استوديو المنبر هو مجموعة أدوات بث مفتوحة المصدر وعالية الجودة، مصممة خصيصًا لصناع المحتوى الإسلامي، والمساجد، والمعاهد التعليمية، والفعاليات المباشرة. تم بناء هذا المشروع بالكامل باستخدام تقنيات الويب الأساسية، مما يوفر نظام تراكب (Overlay) خفيف الوزن وعالي الأداء لبرنامج OBS Studio، ومزود بلوحة تحكم إنتاجية عن بُعد.

## ✨ المميزات
* **12 مشهدًا احترافيًا**: مشاهد مخصصة للبدء قريبًا، البث المباشر، الاستراحة، النهاية، البث متوقف، الإعلانات، الرعاة، الجدول الزمني، الدعاء، القرآن، والمزيد.
* **13 أداة قابلة للتخصيص (Widgets)**: ساعات ديناميكية، عدادات تنازلية، شريط أخبار، صناديق محادثة، أهداف التبرعات، وبطاقات معلومات المتحدث.
* **7 قوالب ألوان (Themes)**: التبديل الفوري بين لوحات الألوان (الذهبي، الزمردي، منتصف الليل، الأزرق الملكي، رمضان، المسجد، والأبيض الأنيق).
* **رسومات متحركة إسلامية**: يتميز بانتقالات مخصصة مثل *تلاشي النور* (Noor Fade)، و*ظهور المشربية* (Mashrabiya Reveal)، و*تفتح الأرابيسك* (Arabesque Bloom).
* **لوحة تحكم الإنتاج**: لوحة ويب عن بُعد للتحكم في البث، وتغيير النصوص فورًا، والتبديل بين القوالب.
* **تخطيطات متجاوبة**: استجابة كاملة بدقة تصل إلى 4K دون أي بكسلة بفضل معمارية SVG.
* **نظام التصدير ZIP**: قم بتصدير وحفظ ملف التراكب المخصص الخاص بك وتوزيعه.

## 🚀 التثبيت وإعداد OBS

1. **المتطلبات الأساسية**: تأكد من تثبيت [Node.js](https://nodejs.org/).
2. **استنساخ المستودع**:
   \`\`\`bash
   git clone https://github.com/alhamodi/minbar-studio.git
   cd minbar-studio
   npm install
   \`\`\`
3. **تشغيل الخادم**:
   \`\`\`bash
   npm start
   \`\`\`
4. **إعداد برنامج OBS**:
   * افتح برنامج OBS Studio.
   * أضف **Browser Source** جديد.
   * عيّن الرابط (URL) إلى `http://localhost:3000/overlay/index.html`.
   * عيّن العرض (Width) إلى `1920` والارتفاع (Height) إلى `1080`.
   * فعّل خيار "Refresh browser when scene becomes active".
5. **فتح لوحة التحكم**:
   * افتح متصفح الويب الخاص بك وانتقل إلى `http://localhost:3000/control/index.html`.
   * يمكنك الآن التحكم في تراكب OBS عن بُعد!

## 📁 نظرة عامة على البنية
* `/overlay`: واجهة التراكب الموجهة لبرنامج OBS. تستخدم وحدات ES Modules لتحميل المشاهد والأدوات ديناميكيًا للحفاظ على الذاكرة.
* `/control`: واجهة مستخدم لوحة تحكم الإنتاج.
* `/server`: خادم Node.js مع بروتوكول WebSocket يحافظ على حالة البث المستمرة (`state.json`).
* `/themes`: تكوينات JSON يتم تحليلها وحقنها كمتغيرات CSS.
* `/assets`: الرسومات المتجهة (SVG)، الانتقالات، والخلفيات.
