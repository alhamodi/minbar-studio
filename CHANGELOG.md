# Changelog

All notable changes to this project will be documented in this file.

## [v1.0.0] - 2026-06-28
### ✨ Features
- **12 Premium Scenes**: Dedicated layouts for Starting Soon, Countdown, Live, BRB, Intermission, Ending, Offline, Announcement, Sponsor, Schedule, Dua, and Quran.
- **13 Widgets**: Fully dynamic and data-bound widgets.
- **7 Themes**: Support for 7 JSON-based themes.
- **WebSocket Control Dashboard**: Added drag-and-drop widget support, live preview iframe, and property inspector.
- **OBS Browser Source Ready**: Responsive scaling up to 4K.
- **SVG Assets**: Native Islamic geometric assets included.
- **JSON Theme Engine**: Swap stylesheets on the fly.
- **ZIP Export**: Download customized profiles from the dashboard.
